Risks to a project

Projects often have alot of chances to fail, failure can come about for several reasons. Time restraints are often a factor of failure since you have a limited time to complete the project, so some parts could end up being unfinished and count as a fail. Adequate planning is also important and ties in with time management since if the project is planned correctly with the time limit in mind there is less room for errors to occur so it can equal a finished project by the deadline.

Legal guidelines

Copyright issues can affect projects with the issue of getting a copyright claim. Using a companies created character or assets without credit to them can get you a copyright claim, another issue you can get is GDPR which is a european law to protect data and privacy. This law can effect projects if a persons personal data is breached if it is used within the project.
