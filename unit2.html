1.5 Describing copy right and constraints

Copyright protect a persons personal work preventing others from claiming it as their own, so cannot be redistributed without their permission.To assure that you do not break copyright laws you have to make sure that you source images that are licensed or copy tight free. There are other aspects that can affect your website such as:
- Users have to provide consent to giving their personal information and that they understand how it is being used
- It must be accessible to everyone to avoid discrimination
- Privacy policy must be displayed to state what information your website is collecting
- Cookie disclosure to show how ur website uses cookies
- Disclose information on your website if it is linked to a business

1.6 Access issues

Websites have to be made with accessiblity in mind to disabled people, so that they are able to comfortably use and understand the website. You can manage this by using a content management system that supports this such as Drupal or Wordpress. Visually management can be sorted by making sure that your website does not contain flashing images (for people with epilepsy), colours that are not used as indicators incase there are colour blind users (or add a colour option for the colourblind), make text larger and more obvious for the visually impaired or add sound indicator and make the website comepletely usable by keyboard alone. 1.7 Appropriate filetypes

The use of appropriate file types for website creation:
- .AVI: This is a standard audio file however you have to download it all before watching
- .CSS: For visual effects on your website that can link to .html
- .DOC: Microsoft word documents that can store text but can only be opened if the word software is owned
- .EXE: Used to open programs for installation
- .GIF: Moving images on a loop
- .HTML: To make the pages for your website
- .PNG: Image file format, can be higher resolution than .jpg also can use transparency
- .JPG: Another form of image file format
- .MP3: Sound file format that is very compressed with good quality
- .PDF: Transferrable document between different platforms
- .RAR: Compressed file format that can contain many other file types
- .TXT: Text file thats easily accessible
- .ZIP: Like .rar files they contain many other file types
